
Full-stack analytics dashboard for evaluating social media post performance using machine learning.

## Features

- Post analytics dashboard
- Linear regression prediction model
- Metrics visualization
- Score ranking
- Best performing post detection
- Historical score analysis
- CSV report export

## Tech Stack

Frontend:
- React
- TailwindCSS
- Axios
- Recharts

Backend:
- Flask
- MySQL
- Scikit-learn

## Machine Learning

Linear regression model predicting engagement score based on:

- Likes
- Comments
- Shares

## Installation

### Backend

pip install -r requirements.txt
python app.py

### Frontend

npm install
npm run dev
